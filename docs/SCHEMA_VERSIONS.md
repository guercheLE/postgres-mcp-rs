The OpenAPI sources retain the complete introspected PostgreSQL catalogs. The embedded stores are publication-focused curated indexes: PostgreSQL 18 retains 500 high-value catalog, monitoring, administration, privilege, and common utility operations; other versions retain the intersection available in that server version. The native, parameterized `execute_sql` tool covers operations omitted from the semantic index.

Every literal input/output schema stored in SQLite is self-contained. Component references are rebased to local `#/$defs/...` references with their transitive definitions embedded alongside the schema. `tests/schema_references.rs` exhaustively checks both SQLite and compressed validation JSON assets for every version.
